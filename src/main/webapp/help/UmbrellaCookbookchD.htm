J. PSI, Entering the DUO User Portal via Umbrella

1. Go to the PSI User Portal

(https://duo.psi.ch/duo/)

2. Click on one of the encircled Umbrella links.

 

 

3. Click on Login with Umbrella

 

4. You are redirected to the Umbrella site

 

5. Login to Umbrella

Go to the middle of the page and log in by entering username and password. ‘Token’ is reserved for future applications and can be ignored at present.

6. Back to and login to the PSI user portal

After logging in to Umbrella you will be automatically redirected to DUO, the PSI user portal.

On the first visit of the PSI User portal as Umbrella user you see the following page:

 

 

If you choose Option 1 you are asked to enter you username/password combination you have used to access the DUO so far.

If you choose Option 2 you will be asked for additional personal information that will  be stored inside the DUO database only. Please note there is no need to choose a username/password (your Umbrella credentials will be valid once the sign-up is completed).

 

 

K. PSI, Logout from the User Portal when logged in via Umbrella

1. You are logged in to PSI User Portal with Umbrella

Just use the logout link in the upper right corner.

After the DUO Logout you see a button to logout you Umbrella account.

 

 

L. Create your Umbrella account from existing DUO User Portal account (Umbrellify)

1. Login to the PSI User Portal

Use your standard DUO User Portal credentials.

 

 

 

 

 

2. Choose Umbrella account creation

On the “Account settings” page choose “Create umbrella Account”:

 

3. Create your Umbrella account

Fill in the missing fields and click on Create Account. Once you have validated your account by clicking on the link supplied in the validation mail you will be re-directed to the ESRF User Portal.
